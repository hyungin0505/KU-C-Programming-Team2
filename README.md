2. `Visual Studio 2022 Community` 실행
3. Github 계정 추가 (Visual Studio에 Github 계정 연동)
4. `Clone a repository (리포지토리 복제)` 선택 후 Enter
5. 레포지토리 경로 설정: https://github.com/hyungin0505/KU-C-Programming-Team2/
